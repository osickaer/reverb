# Reverb - 2026-03-16

## Project Overview
**Reverb** is a daily knowledge fitness app designed to help adults identify and strengthen weak spots in real-world knowledge (history, geography, civics, etc.) through short, personalized quiz sessions. 

- **The Problem:** Adults want a private, low-friction, enjoyable way to fill embarrassing gaps in baseline knowledge without feeling like they are back in school or just tapping through mindless trivia.
- **The Solution:** A daily "knowledge workout" aiming to improve recall and breadth of knowledge, featuring:
  - A weak-spot identification model
  - A structured daily learning loop
  - Adaptive retrieval practice with quick, memorable explanations
  - A progress tracking system that makes users actually feel sharper over time

## Implemented So Far
- **App Structure**: Updated Expo Router structure to use a two-tab baseline.
- **Main Tabs**:
  - `Home` (`app/(tabs)/index.tsx`): Now features a routine-based "Today's 5" dashboard. Shows session status (Available, In-Progress, or Completed).
  - `Progress` (`app/(tabs)/progress.tsx`): Placeholder for tracking knowledge growth.
- **Quiz Flow & Logic**:
  - Fully implemented the question-answer-feedback loop.
  - `QuizScreen` (`app/quiz.tsx`): Dynamically renders questions from the daily session set.
  - `AnswerFeedbackScreen` (`app/answer-feedback.tsx`): Evaluates index-based answers, provides immediate feedback/explanation, and tracks misses.
  - `SessionSummaryScreen` (`app/session-summary.tsx`): Displays final score and session overview.
- **Local Persistence & Daily Routine**:
  - Integrated `@react-native-async-storage/async-storage` for local state tracking.
  - Created `utils/storage.ts` to manage `DailySession` and `UserStats`.
  - Implemented "Today's 5" logic: picks 5 random questions daily, avoiding recently seen ones.
  - Session state persists even if the app is closed (allows resuming "In-Progress" sessions).
- **Seed Data & Schema**: 
  - Updated `Question` interface to include `correctIndex`, `subdomain`, `difficulty`, and `tags`.
  - Expanded `data/questions.ts` to 25 seed questions across multiple domains (Science, History, Geography, Tech, Math).
