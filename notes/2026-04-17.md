# Reverb - 2026-04-17

## Session Summary

Today we added the first version of daily reminder notifications and created a new Profile area to hold personalization and settings. The reminder feature is intentionally lightweight: users can opt into a 7:00 PM local reminder that only fires when the daily session has not been completed.

## Implemented Today

- **Daily Session Reminder Notifications**:
  - Added `expo-notifications` and a small notification helper for scheduling and canceling local reminders.
  - Scheduled a same-day 7:00 PM reminder only when the current daily session is incomplete.
  - Canceled the pending reminder as soon as the daily session is completed.
  - Updated the reminder notification copy to feel friendlier: `Reverb check-in 🧠` with a short daily-workout nudge.

- **Notification Permission Flow**:
  - Added a friendly startup pre-prompt when notification permission is still undecided.
  - Only triggers the native system permission prompt after the user chooses `Turn on reminders`.
  - Persisted the user's daily reminder preference so reminders are opt-in instead of being requested silently.

- **Profile and Settings Structure**:
  - Replaced the top-level `Settings` tab with a `Profile` tab.
  - Added a Profile landing screen with a `Settings` menu item for future profile customizations.
  - Moved notification controls and debug tools into a dedicated `profile-settings` stack screen.
  - Put the longer notification explanation behind an info button next to the daily reminders toggle.

- **Notification Testing Tools**:
  - Restored debug controls, now under Profile -> Settings.
  - Added a test notification button that schedules a local notification 5 seconds out.
  - Added a daily reminder logic test button that exercises the same incomplete-session check without waiting until 7:00 PM.

- **Settings UI Polish**:
  - Updated the notification toggle to use a more native iOS-style contrast treatment in light and dark mode.
  - Used a neutral gray off state, green on state, and white thumb so the control no longer blends into the card background.
